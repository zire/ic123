## Certified Query：可认证查询
### 中文释义
**可认证查询**是其响应经过认证的查询调用。
### Wiki Definition
A **certified query** is a query call for which the response is certified.  [English definition link](https://wiki.internetcomputer.org/wiki/Glossary#catch-up_package_.28CUP.29)
